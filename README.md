# Bank Management System
A console-based **Bank Management System** developed in **C++** using **Object-Oriented Programming (OOP)** concepts and **file handling**. The project simulates basic banking operations while storing data persistently in text files.

---

## Features

- Customer Management
  - Create new customers
  - View customer information

- Account Management
  - Create bank accounts
  - Display account details

- Banking Operations
  - Deposit money
  - Withdraw money
  - Transfer funds between accounts

- Transaction Management
  - Record transactions
  - View transaction history

- File Handling
  - Persistent storage using text files
  - Data remains available after the program closes

---
